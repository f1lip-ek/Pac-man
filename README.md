# Pac-man
Final project for my second year of Computer Science.

It is a simple game inspired by the famous game **Pac-man**.

---
## Controls
| Action   | Key    |
|----------|--------|
| Go up    | W or ↑ |
| Go down  | S or ↓ |
| Go right | D or → |
| Go left  | A or ← |
---
## How to run
To start the .jar file, you need SDK level 24 or higher.

link to download [Java 24](https://www.oracle.com/java/technologies/javase/jdk24-archive-downloads.html)

You can do double click on the .jar file to start it, or you can run the .jar file from the command line
```bash
java -jar ./Pac-man.jar
```
